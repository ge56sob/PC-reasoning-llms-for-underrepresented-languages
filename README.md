# PC-reasoning-llms-for-underrepresented-languages
"The aim of the project will be to test and improve math reasoning capabilities of LLMs of various sizes for many languages -- possibly, including yours!

The project can go in various ways: (i) either we simply select a set of languages and then test models on them and improve them; (ii) or we can start creating such datasets for your language and then bechmarking models. The pipeline will be to translate PolyMath data https://huggingface.co/datasets/Qwen/PolyMath and then manually check it for your native language(s).

Possibly, the project will intersect with our upcoming LLMs with limited resources shared task https://www2.statmt.org/wmt25/limited-resources-slavic-llm.html where we will try to improve cross-lingual math reasoning via improving machine translation capabilites of the models."
